# Roadmap from Web MVP 1.0.0 to 1.0 candidate

## 1.0.0: data and export hardening

- Backend CSV import endpoint with column mapping and import-quality summary.
- JSON export/import round trip for `FitResult`.
- Parameter CSV export.
- Browser-side plot export preset.

## 1.0.0: model-builder completeness

- Reorder components.
- Preset library.
- Better custom-function editor with validation preview.
- Equation preview side panel.

## 1.0.0: fitting parity

- Multi-start / staged seeding.
- More residual weighting modes.
- Explicit compliance detection settings.
- Regression tests against Tkinter backend sample outputs.

## 1.0.0: multi-trace workflow

- Load multiple traces.
- Fit selected trace or batch.
- Compare fits.
- Shared/fixed parameter strategies.

## 1.0 candidate

- Stable ModelSpec/FitResult schema.
- Reproducible reports.
- Windows local launcher or desktop wrapper decision.
- Documentation for users and agents.
- Performance and regression gates.
