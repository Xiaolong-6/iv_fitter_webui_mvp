# Publication Data

Reserved for future publication-derived CSV files.

Any file placed here should include enough provenance for responsible reuse. Prefer a sidecar `.meta.json` file next to each CSV:

```json
{
  "title": "",
  "source": "",
  "citation": "",
  "license": "",
  "notes": ""
}
```

Publication-derived data may also be copied into `demo_data/iv_traces/` when it is suitable as a user-facing demo trace.
